# Car Showroom Management System

A web-based Car Showroom Management System built using PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap.

## Features

- User Login & Registration
- Admin Dashboard
- Car Listings
- Booking System
- MySQL Database
- Responsive Design

## Installation

1. Install XAMPP.
2. Copy the project into the htdocs folder.
3. Import the database.sql file into phpMyAdmin.
4. Start Apache and MySQL.
5. Open:
   http://localhost/Car_Showroom_Management/

## Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap
